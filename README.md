#Tecnologias utilizadas
- Framework: React.js
- Gerenciador de estado: zustand
- Estilização: Tailwind
- Roteamento: React Route
- Build: vite

#Estrutura do Projeto
```text
├── src/
  ├── assets/ # Imagens, fontes, ícones
  ├── components/ # Componentes reutilizáveis
  ├── context/ # Gerenciador de estados
  ├── navigation/ # Navegação da página
  ├── pages/ # Páginas da aplicação
  ├── routes/ # Roteamento
  ├── types/ # Tipo de dados
  ├── api.ts # Conexão com api  
  └── main.tsx # Componente principal
```

# Como executar o projeto
## Pré-requisitos
- backend: [backend](https://github.com/alexsanderkafka/test-stage-backend)
- NodeJs >= 22.22.0
- npm

## Instalação
```bash
# Clonar o repositório
git clone https://github.com/alexsanderkafka/test-stage-frontend.git

# Entrar na pasta do projeto
cd test-stage-frontend

# Instalar dependências
npm install
```

## Executar em desenvolvimento
```bash
npm run dev
```
